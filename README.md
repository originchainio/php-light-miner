Simple mining software based on PHP


Usage:

Solo mining: php ./Lightminer.php "solo" "node" "public_key" "private_key"
  
Pool mining: php ./Lightminer.php "pool" "pool" "your-address" "your-work"
  
=================================================================
  
PHP environment:

version > 7.2

need:  gmp,openssl,PASSWORD_ARGON2I
